Welcome to Only The Recipe

No ads, no life stories. Just the recipe — saved locally, available any time.

How to use it

1

Visit any recipe page

Only The Recipe automatically detects recipe content using structured data and page signals. No setup needed.

2

Click the button

A 🍽 Only The Recipe button appears in the bottom-right corner of the page. Click it to collapse everything except the recipe.

3

Save it to your library

Click Save recipe inside the clean view. The recipe is stored locally — no account required.

4

Browse your library anytime

Click the Only The Recipe icon in the Chrome toolbar to see all saved recipes, add notes, or open a cached read-only view.

5

Restore the page when you're done

Click ✕ Show full page to instantly restore the original layout. No reload, no lost scroll position.

What's included

Zero accounts
Everything lives in Chrome's local storage. Nothing is sent anywhere.
Live view
Inherits the site's fonts and colors so the recipe feels at home.
Personal notes
Add freeform notes to any saved recipe from the popup.
Cached view
Open a clean standalone copy of any saved recipe, even offline.
Reversible
The clean view is non-destructive. Restore the page in one click.
Extensible
Add new sites with a one-line entry in plugin.json.
